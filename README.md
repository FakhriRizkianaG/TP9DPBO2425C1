# Penjelasan Program
Aplikasi ini adalah contoh sederhana implementasi pola Model-View-Presenter (MVP) untuk mengelola data Pembalap dan Tim balap. Program ini bertujuan untuk memisahkan logika bisnis, tampilan, dan interaksi pengguna, sehingga kode menjadi lebih terstruktur, mudah diuji, dan dipelihara.

<h3>Fitur Utama</h3>
<ul>
  <li>Manajemen Entitas Ganda: Mendukung dua entitas utama: Pembalap dan Tim.</li>
  <li>CRUD Lengkap: Mendukung operasi Create (Tambah), Read (Baca/Tampil), Update (Ubah), dan Delete (Hapus) untuk kedua entitas.</li>
  <li>Routing Terpusat: Menggunakan Index.php sebagai router untuk mengarahkan permintaan ke Presenter yang sesuai.</li>
  <li>Arsitektur MVP: Pemisahan tugas yang jelas antara Model (mengelola data), View (mengelola tampilan), dan Presenter (menangani logika interaksi).</li>
</ul>
